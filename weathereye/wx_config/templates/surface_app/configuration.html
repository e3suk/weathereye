WIS2 in a Box (wis2box) is a free and open-source tool that serves as a reference model for a WMO WIS2 Node. It provides an easy-to-use set of tools to collect, process, and share weather, climate, and water data using standardized methods aligned with WIS2 principles. Wis2box also allows access to all data within the WIS2 network and is designed to be accessible for data providers, offering essential tools and services for data discovery, access, and visualization
